# Contributing

Thank you for helping make Liuyao easier to explore in modern AI products.

## Principles

- Keep the casting engine deterministic.
- Let AI explain structured data, not invent chart data.
- Treat Liuyao with cultural respect.
- Avoid claims of certainty.
- Keep integrations simple enough for agent builders to copy.

## Development

```bash
npm test
npm run demo
```

The visual demo is static:

```txt
demo/index.html
```

## Pull Requests

Good contributions include:

- More verified Liuyao rules
- Better English terminology for international users
- More LLM integration examples
- UI improvements that preserve the calm oracle experience
- Tests for casting and interpretation payloads

Please include tests when changing casting logic.
