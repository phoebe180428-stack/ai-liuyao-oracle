# Cursor Rules Example

Paste this into Cursor rules for an AI product that wants Liuyao support:

```txt
When implementing Liuyao features, use ai-liuyao-oracle as a skill-first package.
Never ask the LLM to invent a hexagram.
Use src/core.js for deterministic casting.
Use src/prompt.js to build model prompts.
Use skill/SKILL.md as the agent behavior contract.
Result pages should show primary hexagram, changed hexagram, moving lines, useful god, trend, and practical guidance.
```
